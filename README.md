# FP Launchpad Website

Official website for FP Launchpad - A Centre for Functional Systems Research and Education at IIT Madras.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000`

## Deployment

This site is designed to be hosted on GitHub Pages:

1. Push to GitHub repository
2. Enable GitHub Pages in repository settings
3. Configure custom domain: fplaunchpad.org
4. Add CNAME record pointing to GitHub Pages

## Site Structure

- `/` - Home page with Vision and Charter

## Customization

Edit `_config.yml` to update site metadata and navigation.
